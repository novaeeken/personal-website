OVER MIJ


Ik ben een beginnende full stack JavaScript developer met een achtergrond in UX / UI design.

Ik ben opzoek naar een plek waar ik iets kan bijdragen, kan leren en kan groeien. Ik heb een liefde voor JavaScript en niets geeft me een grotere kick dan wanneer ik een probleem heb opgelost met code. Als het niet moeilijk is, vind ik het niet leuk.

Language-wise voel ik me het meest thuis bij Node.js/ Express, maar ik wil nog veel meer leren. Van leren wordt blij en daarom pak ik dingen snel op. Innovatie geeft me energie. Daarnaast ben ik een

natuurlijke spreker en neem ik graag initiatief. Ik kan razend enthousiast zijn over dingen die ik belangrijk vind en steek mensen daar snel mee aan. In mijn browser staan altijd 34 tabbladen tegelijkertijd open maar ik weet precies wat waar staat. En hoewel ik (nog) geen ster ben in tafeltennis, kun je wel me ontzettend blij maken met een stevige bootcamp-sessie in het park of een lekkere koffie verkeerd.
