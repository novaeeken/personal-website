WEB APPLICATIE

Front-End | Back-End



Foodz: food blogging centralized. Dit is een fictief blogplatform waar iedereen een profiel kan aanmaken en naar hartenlust recepten, tips en weetjes kan posten. Deze opdracht was een technische oefening om complexere applicaties te leren bouwen die schaalbaar zijn.

Bekijk dit project op Github

Technologies used

Node.js, Express, PostgreSQL database, Sequelize en Sessions, Bcrypt, RESTful routing, JavaScript, jQuery en AJAX, Bootstrap v4, Pug en CSS3 in een MVC framework.

Focus op de back-end


Deze opdracht was een technische oefening om complexere applicaties te leren bouwen die schaalbaar zijn. Alle account-informatie, blogposts en comments moesten daarom worden opgeslagen in een relationele database (PostgreSQL). Hierbij is bewust gebruik gemaakt van een MVC framework. Pagina’s worden gevuld met content op basis van specifieke http requests naar de server.

Functionaliteit

Het platform geeft gebruikers de mogelijkheid een profiel aan te maken (met profielfoto en beschrijving), in- en uit te loggen, blogs te posten, blogs te liken, en reacties te plaatsen onder posts van anderen. Wachtwoorden worden versleuteld met behulp van Bcrypt. In deze Node.js applicatie is tevens gebruik gemaakt van RESTful routing en dynamic parameters die de pagina’s bevolken met de juiste informatie.
